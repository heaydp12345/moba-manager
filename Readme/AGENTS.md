# AGENTS.md

# 專案資訊

專案名稱：

moba manager(暫定)

遊戲類型：

電競經營模擬 + MOBA 比賽模擬器

參考作品：

* Teamfight Manager 2
* League of Legends
* TFT 裝備系統

---

# 最終目標

製作一款電競經營遊戲。

玩家將經歷以下階段：

第一階段：

職業選手

第二階段：

職業生涯結束

第三階段：

成為教練或經理

第四階段：

經營電競戰隊

最終玩法包含：

* 招募選手
* 培養選手
* 戰術規劃
* BP 系統
* 戰隊管理
* 聯賽競爭
* 世界賽

最終階段玩家不直接控制英雄。

比賽完全由 AI 自動進行。

---

# 開發原則

優先建立基礎系統。

開發順序：

1. MOBA 比賽模擬器
2. AI 系統
3. 選手生涯系統
4. 戰隊經營系統
5. UI
6. 優化與平衡

禁止在模擬器未完成前開始開發經理模式。

MOBA

* 紅方主堡
* 上路
* 中路
* 下路
* 野區
* 防禦塔

* 尋路
* 出兵
* 野怪

# 英雄系統

英雄資料來自 hero.json。

每位英雄必須支援：

* 名稱
* 定位
* 基礎能力
* 成長能力
* 技能
* 攻擊類型
* 射程
* 移動速度

定位類型：

* 坦克
* 戰士
* 刺客
* 法師
* 射手
* 輔助

禁止硬編碼英雄。

---

# 技能系統

技能資料來自 skill.json。

每位英雄擁有：

* 被動
* Q 技能
* W 技能
* E 技能
* R 技能

技能需支援：

* 傷害
* 治療
* Buff
* Debuff
* 控制
* 投射物
* 範圍效果

# 裝備系統

參考 TFT 設計。

英雄最多裝備：

3 件

散件 + 散件 = 成裝

散件數量：

9 件

具體參考item.xlsx跟item.txt
